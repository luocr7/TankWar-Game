# TankWar-Game
基于多线程的坦克大战游戏
======================

主要涉及
* c#绘图及键盘事件绑定(包括重新绘制及双缓冲解决闪烁问题，利用公共变量解决键盘阻塞监听问题)
* 多线程任务的调度(考虑线程安全性)
* 多线程任务的调优(选择适当线程休眠时间进行优化)

重点内容
* 多线程任务调度及调优
